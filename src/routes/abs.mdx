对于无符号整数类型，右移高位补0。

对于有符号整数类型，高位**补符号位**。

```rust
fn main() {
    let x: i32 = -1;
    let result = x >> 31;
    println!("{result}");  // 输出: -1
    let y: i32 = 1;
    println!("{}", y >> 31); // 输出: 0
}

```

对于负数，右移 31 位全部填充符号位1得到 -1 即 0xffffffff
