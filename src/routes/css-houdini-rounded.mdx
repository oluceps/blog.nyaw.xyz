破事水

如果你用的是先进的 Chromium 浏览器，可能已经注意到了右上角的navigator bg和往下滚会出现的
back-to-top 按钮，它们的圆角比常见的css圆角更圆润一点：

一般圆角：
<div class="h-10 w-10 rounded-lg bg-cerise-300"/>


好看圆角,
当然这个的浏览器支持[很有限](https://caniuse.com/?search=houdini)，比如你是火狐就看不出区别(fallback了)：
<div class="h-10 w-10 squircle rounded-lg bg-cerise-300"/>

因为前者是简单的1/4圆弧拼接而后者是~~G2曲率连续的图形拼接~~没看实现但是应该更复杂一点，详细区别可以看[b站](https://www.bilibili.com/video/av979946928)，总之过渡要平滑不少。
