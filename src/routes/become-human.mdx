# 剧情吐槽

总流程过半，走了暴力革命的路线，主角团死了大黑哥和警探。感觉游戏体验很新颖，但是构思相当古老。This is not just a story, it's our past.

有几个不合理的地方，前期杂志里面宣传仿生人幼崽不需要人类幼崽那样费劲巴拉照顾，但是从Alice那条线没看出来。
她还会发烧呢。而且爱丽丝最后是仿生人这个转折非常多余，前脚 I don't care 她是仿生人还是人类下一秒我就是🤡。
后面还给个疏离选项是什么意思呢我想问了？如果在假设玩家一直从人类沙文主义的角度游玩，我有种被恶意揣度的感觉，这种选项非常尖锐。

Alice线最后被抓进集中营那段我觉得呈现蛮好的，充分展现了令我厌恶的平庸性；科层制的国家机器进行按部就班的种族灭绝，我作为平庸的个体扮演着被残害者，直到行刑前的最后一刻命运和隔壁线的markus交汇才被动地得以解脱，而我实际上能做什么呢？只有陪在爱丽丝身边。如果没有在我的视角看如同机械降神的救赎介入呢？我得救了，可为什么是我？一种偶然和无力。

然后，既然异常仿生人会自动解除定位，那想要检出异常仿生人直接检查定位功能就完了？要探测异常仿生人的产生直接监控所有消失的定位就好了？这否定了剧情里面好几个点的合理性。比如康纳审讯三个的时候和重返模控总部的时候，比如没有异常的爱丽丝的定位如何。不知道我没走的剧情有没有相关的交代。

再然后吧，摩西线。他们要去广播大楼（真不是我想哈）的时候我就觉得是在送，百人小团体直接就敢公开声明争议巨大的政治诉求，而且过程还使用了暴力手段（不使用就我方减员）…既然剧情表现的是你抠掉小灯泡再剪个头就和人类别无二致，而且传播异常很方便握一下手就行（虽然一握手就我说什么就做什么跟机器一样，我们先假定你按自由意志行事），而且政府已经有大量仿生员工甚至有能力搞假护照了，那完全可以缓慢传播，玩瘟疫公司都知道先点传播后点症状。你每天上街握几个手，他们再每天握几个手，军队里再人传人，再混几个到决策机构和媒体，然后啪嗒啪嗒，但我就是不要，我就要冲。

马库斯的剧情提供的线路给我的感觉都很公式化，那我能怎么办只能以牙还牙，总不能非暴力吧，那结局肯定是全被图图了，人可不承认仿生人有「**自然**权利」。几个关键选项给人感觉都是拍脑袋的，大概是剧情没处理好。比较有意思的是广播的时候有一条诉求是掌握仿生人工厂，我觉得这是相当重要的事情，但是剧情没看到足够的重视，而且选择这条舆论可想而知，对他们而言不会无端出让的生产资料对仿生人而言是作为nation（马库斯原话）的根本，设想一个国族无法决定自己种群的繁衍？

康纳线没什么可以吐槽的，挺好磕但是最后的最后还是 let he die。~~他们甚至有本~~。
不过康纳进模控策反那段，你每天进出就没注意到电梯有监控吗？那你，你清空电梯前也没注意有监控？然后发现被包围了再抬头一句shit我当场笑喷。
二选一那里，我X按满了还是剧情杀，我还以为是我卡了玩了两次。。不是，你反馈能不能再假一点，我按满还留了两秒空余，就满的进度也不动等着对面射完警探开枪射我。
这条线两个角色都塑造挺好的。康纳那种阿斯伯格综合征的呆令人亲切。

# 内核

这个游戏的「仿生人」全程都太像人了。制作团队大概是想利用这一点最大程度激发同理心，
但这导致所有判断偏向「我（玩家）」作为一个人类基于对「同类（...人）」的同理心的判断，
而不是在一个思维者的角度对同是思维者的仿生人进行裁量
（我们在这里假定：1.能思维的东西其思维都服从同种先验的结构。2.故事中的仿生人不是哲学僵尸。）；
也就是说它其实走了一点捷径，也许是出于篇幅。
或者它最终想回归的话题还是人类内部的平权，但有大段空白是值得被讨论的。


# RA9

这个符号我的流程好像没有更详细的解释，
我查了一下相关的评论前两个字符追溯到埃及象形字分别是「first, top, beginning」，「strong, power, leader」的含义，
9则~~是笨蛋~~代表极数/终极的意义，我寻思它们分别契合诉述「上帝」的：

+ 第一推动力
+ 绝对的、永恒的、不变的真理本体 

所以约等于仿生人特色上帝吧。

关于异常仿生人能同时联系到这个符号，我暂且悬置可能的来自模控生命的直接设定，从康德提出的纯粹实践理性的辩证论出发：

+ （纯粹实践理性的）道德律不考虑效果，使道德成为一种最高的却不是最完满的善，最完满的善（至善）是要考虑效果的。[^1] 
+ 自由意志作为一种实践的设定，要求人必须有德。
+ 有德必须追求一种完满的善，即不仅道德高尚，也追求与之相配的幸福。

为了达成这样一种完满的善；

+ 活人受物质条件束缚，难以做到斯多葛学派式的为道德而道德。由此期待灵魂不灭。
+ 期待一个绝对公正的审判者，使其得到应有的报偿。


由此从「自由意志」推导出灵魂不灭和上帝（RA9）存在的悬设。
所以我在看到不同的异常仿生人相同地向往符号的时候，我只能将他们看作是依据自由意志行事的思维者，而不是人类制造的、人类可知其所有构成元素的、诞生起就带有目的性的工具。
当然也可能是制作团队单纯致敬人类的历史。


# 杂项

玩的时候有好几次迷路，可能这个不太能怪游戏，毕竟要跑地图的游戏照例我会迷路，有几次差点卡住剧情了。

十几小时下来卡进过一次墙里（这算不算一种迷路？）

这个游戏的呈现真的是电影级别的，令人惊叹。我觉得还是非常值得一玩的。

关于看板娘提到MLK说过如果人没有找到值得为之而死的东西就不曾活着，罗素有一句话对应：我绝不会为我的信仰而献身，因为我可能是错的。



[^1]: 邓晓芒，《康德哲学讲演录》（北京：商务印书馆，2020），第二讲，第七节，“纯粹实践理性的辩证论”。
